# 🏭 Production Planning & Resource Balancing Tool

## 📌 Overview
This project is a production planning and resource balancing tool designed to identify inefficiencies, detect bottlenecks, and improve decision-making in industrial processes through data analysis.

---

## 🎯 Objectives
- Monitor resource utilization across production phases  
- Compare production capacity vs workload  
- Identify bottlenecks and critical constraints  
- Support data-driven resource allocation decisions  

---

## 📊 Key Features

### 1. Resource Management
- Operator database  
- Skill mapping (levels 1–4)  
- Assignment to primary and secondary phases  
- Operator status (active, on leave, in training)  

### 2. Production Analysis
- Cycle time per phase  
- Production capacity per shift  
- Workload (orders)  
- Utilization rate calculation (%)  

### 3. Bottleneck Detection
- Automatic identification of bottlenecks  
- Risk detection for highly saturated phases (>90%)  
- Global KPI monitoring  

### 4. Decision Support
- Resource reallocation suggestions  
- Data-driven operational insights

## ⚠️ Case Study Example
- **Critical phase:** Welding (F4)  
- **Utilization:** 164%  
- **Issue:** Workload exceeds capacity (210 vs 128)  
- **Suggested action:** Reallocate one operator from a lower-utilization phase

## 🛠️ Model Structure

### Inputs

- Production workload  

### Outputs
- Production capacity  
- Utilization per phase  
- Bottleneck identification  
- Suggested corrective actions
